# DashboardCompanion

Small SwiftUI iPhone companion app for the team's NT4 dashboard.

Open `DashboardCompanion.xcodeproj` in Xcode, pick your signing team, and run on an iPhone.

The app packages:
- `dashboard.html`
- `field.png`

It loads the same dashboard UI in a native `WKWebView`, saves the last robot IP, and can reconnect quickly from the top bar.
